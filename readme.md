# Tela de Login

Projeto de uma tela de login moderna desenvolvida com HTML e CSS, seguindo o tutorial [Projeto Tela de Login com HTML e CSS](https://www.youtube.com) do YouTube.

## Sobre

Interface de login com design moderno e efeito glassmorphism (vidro fosco), ideal para praticar HTML e CSS.

## Tecnologias

- HTML5
- CSS3
- Google Fonts (Poppins)
- Boxicons

## Como usar

1. Clone o repositório
2. Crie uma pasta `img` e adicione uma imagem de fundo chamada `img.jpg`
3. Abra o arquivo `index.html` no navegador

## Estrutura

```
projeto-login/
├── index.html
├── styles.css
└── img/
    └── img.jpg
```

## Funcionalidades

- Campos de usuário e senha
- Checkbox "Lembrar senha"
- Link para recuperação de senha
- Botão de login com animação
- Design responsivo
